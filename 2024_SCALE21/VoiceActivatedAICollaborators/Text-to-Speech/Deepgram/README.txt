Requirements:
For using Deepgram's Text-to-Speech, you will need an account which you can sign up for a free at: https://deepgram.com


Steps to run summarization:
1. [recommended but not required] Create a Python Environment using something like Conda
2. set your DEEPGRAM_API_KEY as an environment variable
3. pip install -r requirements.txt
4. run the "hello world" script: python main.py
